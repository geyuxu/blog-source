nodejs操作文件
发布于：2016-03-18 00:00:00 标签：/ nodejs / 访问：

向文件中写数据：

var txt = '保存的数据';
fs.writeFile('message.txt',     
     txt, (err) => {
        if (err) throw err;
        console.log('It\'s saved!');
    }
);

fs.writeFile的同步方法是

fs.writeFileSync(file, data[, options])

从文件中读取数据：

fs.readFile('/etc/passwd', (err, data) => {
    if (err) throw err;
    console.log(data);
});
