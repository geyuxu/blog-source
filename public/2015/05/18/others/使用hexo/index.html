准备工作

安装nodejs

安装hexo

1.安装hexo-cli

npm install hexo-cli -g

2.初始化blog文件夹

hexo init blog

如果当前文件夹是空文件夹，也可以省略“blog”。



3.用npm安装依赖包

cd blog
npm install

4.启动hexo服务器，预览效果

hexo server

用浏览器打开http://localhost:4000

恭喜你，拥有了自己的网站:)

用hexo写一篇日志

输入命令在source_posts目录下新建一个markdown文件

hexo new 服务端工程师入门与进阶Java版

用markdown编辑器打开文件，并编辑。

输入

hexo g
hexo s

在网页中查看效果
