npm使用国内镜像
发布于：2015-05-28 00:00:00 标签：/ nodejs / npm / registry / 访问：

切换方法

方法一：在命令上指定

npm --registry http://registry.npm.taobao.org

方法二：通过config命令

npm config set registry http://registry.npm.taobao.org

方法三：编辑 [User Home]/.npmrc 加入下面内容，此方法同方法二等效

registry = http://registry.npm.taobao.org

国内镜像

淘宝镜像：http://registry.npm.taobao.org

cnpmjs：http://registry.cnpmjs.org

npm publish

npm publish命令会发布你自己的模块到npm社区，所以在使用该命令前要切换回原先的registry：https://registry.npmjs.org
